FedoraRelsExt

Extends \FedoraRelationships

This is the base class for Fedora Relationships.

This code lets the user define a new method that will be called when a property is accessed. Any method that ends in MagicProperty is code that implements a magic property.

The object for the relationship to filter by.

mixed

What the attribute type should be. One of the defined literals beginning with RELSTYPE. For backwards compatibility we support TRUE as RELS_TYPE_PLAIN_LITERAL and FALSE as RELS_TYPE_URI.

Response

array
