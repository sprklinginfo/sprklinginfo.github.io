getRequest(string $url, boolean $headers_only = FALSE, string $file = FALSE) : array
abstract
throws

Arguments

$url

string

The URL to post the request to. Should start with the protocol. For example: http://.

$headers_only

boolean

This will cause curl to only return the HTTP headers.

$file

string

A file to output the content of request to. If this is set then headers are not returned and the 'content' and 'headers' keys of the return isn't set.

Response

array

Associative array containing:

  • $return['status'] = The HTTP status code
