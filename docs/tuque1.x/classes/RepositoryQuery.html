parseSparqlResults(string $sparql) : array
static

Arguments

$sparql

string

A string containing Sparql result XML.

Response

array

Indexed (numerical) array, containing a number of associative arrays, with keys being the same as the variable names in the query. URIs beginning with 'info:fedora/' will have this beginning stripped off, to facilitate their use as PIDs.

Utility function used in self::query().
